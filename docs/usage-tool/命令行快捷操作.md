# 命令行快捷操作

## 查询文件夹下所有的文件代码行数并排序

```bash
# git-bash命令
git ls-files | xargs wc -l | sort -nr

# cmd命令
find . -name "*.*" | xargs wc -l | sort -nr
```

## 解除端口占用
<!-- tabs:start -->

<!-- tab:linux -->
```bash
# 查看是否被占用，找到PID信息
netstat -nap

# 特定端口
netstat -nap | grep 8080

# 杀死进程
kill -9 PID

# 查看当前运行的进程，看是否已被杀死
ps
```

<!-- tab:windows -->
```bash
# 查看端口号
netstat -ano | findstr 8080

# 杀死进程
taskkill -pid PID -f
```
