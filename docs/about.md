# 关于本站

## 符号标记

列出本站使用到的符号及其含义：

- ❌：表示实际上和记录有误或不一致的内容    
- 🟢：表示不太熟悉且重要的内容     
- 😢：表示尚未搞懂的内容   
- ⛔：阅读中断标志
- 🔴：重点、易混淆、难以理解的内容

## 内容解释

- 内容可能和实际有出入，在有出入时，应当以实际为准
- 内容可能过时
- 对于前端项目，内容出错大概率是nodejs版本、npm工具类型（npm、pnpm等）、网络环境等导致的
