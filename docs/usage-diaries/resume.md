# resume

## 命名

- `姓名-目标岗位-优势点`
- `姓名-目标岗位`
- 以此类推，尽量精准简要

## 详细项

**个人信息**：

- 姓名
- 性别
- 电话
- 邮箱
- 出生年月
- 工作年限
- 所在地
- 最高学历

**教育经历**：

- 学校
- 学历
- 专业
- 时间
- 可选：成绩、荣誉、奖项

**个人优势&技能**：

- 专业技能
- 可选：个人博客、比赛获奖

**工作经历**：

- 公司
- 部门
- 职位、职级
- 工作内容、成果；工作职责、内容

**项目经历**：

- 公司
- 项目
- 角色
- 时间
- 技术栈
- 详细项目内容（star原则：背景、任务、动作、结果）
